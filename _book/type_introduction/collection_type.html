集合型別

集合型別

有機會再補...

  1. Array
* Array 陣列
* 一個`有序`的資料集合，利用 Index 來取出陣列中的值
* 利用 , 來區分每一組數值
* ex: let arr = [0, 1, 2]
* arr[0]
* arr[1]
* note: 存放的內容，必須明確的讓 swift 知道：陣列中存放的型別是什麼
  1. Dictionary
* Dictionary 字典
* 一種`無序`的資料集合，是一組 key value
* 用 : 區分 key valueㄝ，左邊是 key，右邊是 value
* 一樣用 , 來區分每一組數值
* key 必須是唯一
* 用 key 來取值
  1. Set
* 一種`無序`的資料集合，跟 Array 很像，但存放的值必須是唯一
